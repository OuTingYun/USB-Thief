## USB Thief
ModeA:Copy the file from USB to `C:/USB`

ModeB:Copy the .py file from desktop to USB

> 資料在電腦與USB間傳輸

### Environment
```cmd
python = 3.6.10
psutil = 5.8.0
```
### How to use
#### ModeA: Copy the file from USB to  `C:/USB` 
- Run main.py
- Input A
- Insert the USB
- All files in USB will be reserve in `C:/` and the files in USB will be hid.
#### ModeB:Copy the .py file from desktop to USB
- Run main.py
- Inupt B
- Insert the USB
- All .py .c .cpp files will be reserved in `USB:/Trophy`
